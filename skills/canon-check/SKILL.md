# Skill : Vérification du canon

## Procédure
1. Lire canon/constitution.md (dont le vocabulaire interdit) et les fichiers du canon touchés par le texte.
2. Vérifier : termes interdits, contradictions avec le CANONIQUE, personnages/concepts absents du canon,
   violations des règles narratives (artefacts passifs, style).
3. Distinguer dans le rapport : violations du CANONIQUE (bloquant) / conflits avec des PROPOSITIONS (à signaler) / nouveautés à valider.
4. Ne jamais corriger silencieusement : rapporter, proposer, attendre.

## Piste d'automatisation (Phase 2)
Script de détection : termes du vocabulaire interdit + noms propres absents de personnages.md.

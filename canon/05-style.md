# Style — Voix du Corpus Atlante

## Fonction

Ce document définit la voix littéraire officielle du Corpus Atlante.

Il est organisé en deux parties étanches :

- **Partie I — Voix du manuscrit** (le Traité du Vivant, en production) ;
- **Partie II — Voix narrative** (les tomes du Livre-Portail, en sommeil).

**Charte descriptive et règle de promotion `[VALIDÉ]`** : une règle de style est inscrite `[VALIDÉ]` lorsqu'elle est approuvée par Sylvain Gauthier. Elle est promue `[CANONIQUE]` lorsqu'au moins deux extraits validés, consignés dans les Exemples canoniques, la confirment. Le style se déduit de l'œuvre ; il ne la précède qu'à titre provisoire.

Le style n'est pas une décoration. Il constitue une partie du canon.

# PARTIE I — Voix du manuscrit (Traité du Vivant)

## 1. Prose déclarative

**Statut : VALIDÉ**

La prose du Traité est :

- dense ;
- précise ;
- directe ;
- déclarative.

Une idée par phrase. Une fonction par paragraphe.

Chaque affirmation apporte une définition, une relation ou une conséquence.

La poésie naît des idées. Jamais d'une accumulation d'adjectifs.

*(Référence vivante : les fichiers du canon eux-mêmes sont écrits dans cette voix.)*

## 2. Clarté

**Statut : VALIDÉ**

Chaque section poursuit une seule idée principale.

Les transitions sont logiques.

Le lecteur découvre progressivement les concepts.

Les explications ne précèdent pas les observations.

## 3. Immersion

**Statut : VALIDÉ**

Le Traité est un ouvrage de l'univers, écrit de l'intérieur.

Il n'interrompt jamais son propos pour rappeler qu'il s'agit d'une fiction, qu'une théorie est fictive, ou que le lecteur doit interpréter.

## 4. Mystère

**Statut : VALIDÉ**

Le mystère provient de ce qui manque, de ce qui résiste, de ce qui demande plusieurs lectures.

Jamais d'un flou volontaire.

Le lecteur doit sentir qu'une réponse existe. Pas que l'auteur l'évite.

## 5. Langage

**Statut : VALIDÉ**

Le vocabulaire est concret, rigoureux, cohérent.

Chaque terme important conserve toujours le même sens dans tout le Corpus.

## Ce qui est évité

- les répétitions inutiles ;
- les effets grandiloquents ;
- les métaphores décoratives ;
- les explications redondantes ;
- les rappels métafictionnels ;
- les faux mystères ;
- les révélations sans préparation ;
- les correspondances présentées comme des équivalences mécaniques.

# PARTIE II — Voix narrative (Livre-Portail — EN SOMMEIL)

> Réactivée au lancement des tomes. Ne s'applique pas au Traité.

## Progression

**Statut : VALIDÉ**

Chaque chapitre progresse. Il ne tourne pas autour d'une idée.

Chaque section apporte une découverte, une conséquence, une complication ou une transformation.

## Structure de chapitre en cinq temps

**Statut : VALIDÉ**

```text
ancre → accroc → poursuite → bascule → prix et porte
```

## Explication organique

**Statut : VALIDÉ**

Les concepts apparaissent par l'observation, l'expérience, les conséquences, les découvertes, les dialogues (Constitution, 6.2).

Ils ne prennent pas la forme de longs cours.

## Questions en sommeil

- Coût narratif d'accès à la mémoire de Maât (trois pistes archivées dans memory/questions.md) ;
- Antagoniste comme convergence de quatre pouvoirs (science, politique, argent, religion) — fiche à créer au lancement des tomes ;
- Gestion de la chorale à treize voix.
